排序-冒泡排序

2022-03-25

算法步骤

比较相邻的元素。如果第一个比第二个大，就交换他们两个。

对每一对相邻元素做同样的工作，从开始第一对到结尾的最后一对。这步做完后，最后的元素就是最大的数。

针对所有的元素重复以上的步骤，除了最后一个。

时间复杂度

最差情况和平均情况时间复杂度是O(n^2),当数组本身有序时是O(n)
根本没啥用

代码实现

以下代码是伪代码:

for (i = 0; i < arr.length; i++) { // 用来比较的指针
	for (j = 0; j < arr.length; j++) { // 用来向后遍历的
		if (arr[j] < arr[i]) { // 如果发现比arr[i]小的，那么就交换位置
			swap(arr[j], arr[i]) // swap函数可以替换，只要能交换就好
		}
	}
}
